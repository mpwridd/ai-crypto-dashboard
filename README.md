# AI Crypto Dashboard 📊

Real-time cryptocurrency market data with AI-powered analysis using **Mimo v2.5 Pro**.

## Features

- 📊 **Live Market Data** — Top 50 coins by market cap (CoinGecko API)
- 🔥 **Top Gainers & Losers** — Real-time tracking
- 🤖 **AI Analysis** — Ask anything about the market
- 📈 **Full Market Table** — Price, 24h, 7d, volume, market cap, search & sort
- 📉 **Stats Overview** — Total market cap, volume, avg change, gainers/losers ratio
- ⚡ **Auto-refresh** — Updates every 30 seconds
- 🔍 **Search & Sort** — Filter by name/symbol, sort by any column

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript** + **Tailwind CSS**
- **Lucide React** (icons)
- **Mimo v2.5 Pro** (AI analysis)
- **CoinGecko API** (market data, free tier)

## Getting Started

```bash
npm install
npm run dev
```

## Environment

```env
MIMO_API_KEY=***
```
